# LeetcodeStackQueue

This repository contains solutions to algorithmic problems from LeetCode focusing on fundamental data structures: **Stack** and **Queue**. 

The main goal of this project is to deeply understand the principles of LIFO (Last-In-First-Out) and FIFO (First-In-First-Out) operations by implementing custom classes and applying them to solve more complex challenges.

---

## Implemented Tasks

### 1. [Implement Queue using Stacks](https://leetcode.com/problems/implement-queue-using-stacks/)
* **Description:** Implement a first-in-first-out (FIFO) queue using only two stacks.

### 2. [Implement Stack using Queues](https://leetcode.com/problems/implement-stack-using-queues/)
* **Description:** Implement a last-in-first-out (LIFO) stack using only two queues.

### 3. [Maximum Frequency Stack](https://leetcode.com/problems/maximum-frequency-stack/)
* **Description:** Design a stack-like data structure to push elements to the stack and pop the most frequent element. If there is a tie for the most frequent element, the element closest to the top of the stack is removed and returned.
